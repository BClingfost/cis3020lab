Fan Creations Page

The Fortnite fandom is pretty large due to the popularity of the game. Because of this size the community has created alot of fan made content.

Fan Made Creative Maps

Fortnite's built in creator portal and unreal editor has alowed its fans to make there own Fortnite modes/maps that premier right inside Fortnite allowing anyone to play them.

  1. TOY BOX - EVERY GUN 1v1

      This map created by Fortnite user Post allows players to team up or fight against each other as they use any gun in the game to try and get xp. The cool part of this map is that it is designed like a gigantic version of someone's bedroom, and the players themselves are the toys.

  2. TOY BED WARS

      3. This map made by Fortnite user Typical Gamer has players face against each other while protecting there homebase and bed upgrading both the longer they survice.

  3. MEGA RAMP SURVIVAL

      4. In this map made by Fortnite user NoahsNoah, players must race to the top of the ramp while also dodging fellow players and falling objects.
